# django-react-app 
# PROJECT-SETUP
# Backend Setup
```cd Back-end```
```pip install django and djanorestframework```
```pip install psycopg2 and also make sure pgadmin is install in your machine```
```python manage.py makemigrations```
```python manage.py migrate```
```python manage.py runserver```
# Frontend Setup
```npm install```
```npm linit```
```npm start```
